# 👨‍💻 Informatika Unima - GitHub Resmi Program Studi Informatika

Selamat datang di **GitHub Resmi Program Studi Informatika, Universitas Negeri Manado (UNIMA)**.  
Repositori ini menjadi pusat dokumentasi, kolaborasi, dan publikasi berbagai proyek mahasiswa, dosen, dan program akademik di lingkungan Informatika UNIMA.

---

## 🎯 Tujuan Repositori Ini

- Menyediakan ruang kerja kolaboratif untuk tugas dan proyek mahasiswa.
- Menyimpan portofolio digital hasil pembelajaran mata kuliah pemrograman dan pengembangan aplikasi.
- Meningkatkan budaya coding terbuka dan praktik penggunaan version control di lingkungan akademik.
- Mempermudah dosen dalam mengevaluasi dan meninjau perkembangan proyek mahasiswa.

---

## 📚 Struktur Organisasi GitHub

Repositori diatur berdasarkan:

- 📂 **Mata Kuliah** (Contoh: `pemrograman-web`, `struktur-data`, `framework-laravel`)
- 📂 **Tugas Mahasiswa** (Contoh: `tugas-akhir-nama-nim`, `project-uas-nama-nim`)
- 📂 **Penelitian dan Pengabdian** (Contoh: `penelitian-ai-2025`, `abdimas-sistem-informasi-desa`)
- 📂 **Template & Panduan** (Contoh: `template-laravel`, `panduan-setup-git`)

---

## ✅ Panduan Mahasiswa

1. **Fork / Clone** repositori tugas yang telah ditentukan oleh dosen.
2. Gunakan struktur folder dan file sesuai instruksi masing-masing tugas.
3. Lakukan `commit` secara rutin dengan pesan commit yang deskriptif.
4. Upload kode dan dokumentasi yang diminta.
5. Pastikan tidak melakukan `push` file yang sensitif seperti `.env` atau password.

---

## 💼 GitHub Classroom

Untuk pengumpulan tugas, kami menggunakan **GitHub Classroom**.  
Mahasiswa akan menerima tautan resmi untuk setiap tugas dan akan otomatis dibuatkan repositori masing-masing di dalam organisasi ini.

---

## 📌 Catatan Etika

Kami menjunjung tinggi **etika akademik**.  
Plagiarisme dalam bentuk apa pun dilarang dan akan dikenakan sanksi sesuai ketentuan akademik.
